# Features

이 서비스가 갖출 기능의 범위(scope). 구현 상태(완료/진행 중)는 추적하지 않음 — GitHub Issues 참조.

용어는 [`glossary.md`](glossary.md), 운영 정책 수치는 [`policy.md`](policy.md) 기준.

> **Platform**: 소비자·사장 클라이언트는 **PWA**, 관리자 클라이언트는 **일반 웹**. 푸시 알림은 **FCM**, 위치는 Web Geolocation API 사용 ([`product.md` Platform](product.md)).

---

## 인증 & 계정

- 회원가입/로그인 — 소비자/사장 유형 분리, 카카오 소셜 로그인
- 본인인증 — 휴대폰 인증, 토스 인증
- 카카오 소셜 로그인 실제 연동 — Mock 구현을 실제 카카오 API 연동으로 교체
- 비밀번호 재설정 — 이메일 링크 발송
- 소비자 프로필 관리 — 닉네임, 사진, 전화번호
- 사장 프로필 관리 — 사장 이름, 연락처, 사업자 정보
- 관리자 계정 생성/초대 — 관리자 셀프 가입 미지원, 기존 관리자 초대 기반
- 주소지 관리 — 최대 3개, 카카오 주소 API
- 알림 설정 — 종류별 On/Off, 반경 설정
- 회원 탈퇴 — 30일 유예, 진행 중 주문 시 제한
- 인증 Rate Limiting — 로그인/회원가입/비밀번호 재설정 요청 제한

## 매장 & 상품

- 매장 등록 신청 — 사업자 인증(국세청 API) + 관리자 승인
- 매장 등록 승인/반려 — 관리자
- 매장 삭제 (사장 요청) — 진행 중 주문 시 제한
- 매장 삭제 승인/반려 — 관리자, 미정산 확인 후
- 매장 정보 수정 — 주소 변경 시 재승인
- 영업시간/휴무일 설정 — 정기/임시휴업
- 일반 상품 관리 — 등록/수정/삭제/품절 처리
- 마감 임박 상품 등록 — 일반 상품을 떨이로 전환, 할인율/수량/마감 시간 설정
- 마감 임박 상품 수정/마감 — 자동 마감(시간 도래) + 수동 마감(사장)

## 탐색 & 검색

- 홈 피드 — 반경 내 매장/마감 임박 상품 카드
- 매장 상세 조회 — 정보 + 상품/마감 임박 상품 목록
- 상품 상세 조회 — 사진, 가격, 할인율, 리뷰
- 즐겨찾기 관리 — 등록/해제/목록
- 지도 기반 매장 조회 — 카카오맵, 핀 표시
- 키워드 검색 — 매장명/상품명/해시태그
- 검색 기록 관리 — 최근 10개, 자동완성
- 실시간 급상승 키워드 — 1시간 내 검색량 기준
- 필터/정렬 — 거리, 할인율, 마감 임박

## 주문 & 결제

- 장바구니 관리 — 매장 혼합 경고
- 주문 생성 — 픽업 시간 지정(15분 단위), 요청사항
- 결제 — 토스페이 API, 포인트/쿠폰 동시 적용
- 주문 취소 (소비자) — `주문접수` 상태에서만 가능
- 주문 수락/거절 (사장) — 거절 시 자동 환불
- 주문 상태 변경 — `주문접수 → 준비중 → 준비완료 → 픽업완료`
- 노쇼 처리 — 픽업 시간 초과 시 처리 (누적 기준·제재 강도는 `policy.md` 참조 / 일부 TBD)
- 주문 목록/상세 조회 — 소비자/사장 각각
- 환불 처리 — 정책은 `policy.md` 참조

## 알림

- 마감 임박 상품 등록 알림 발송 — 우선순위: 현재 위치 → 즐겨찾기 → 설정 주소
- 마감 임박 상품 알림 수신 (소비자) — 3가지 우선순위 채널
- 주문 관련 알림 — 신규 주문, 상태 변경, 취소, 환불
- 매장 관련 알림 — 승인/반려, 삭제 승인/반려
- 리뷰 관련 알림 — 신규 리뷰, 답글
- 혜택 알림 — 포인트/쿠폰 소멸 예정
- 마케팅 알림 — 이벤트/쿠폰
- 운영 알림 — 회원 제재

## 리뷰 & 소통

- 리뷰 작성/수정/삭제 — 별점 + 텍스트 + 사진 3장, 수정은 7일 이내
- 리뷰 목록 조회 — 최신순/별점순/사진만 필터
- 리뷰 답글 (사장) — 공개 답글
- 리뷰 신고 — 부적절 리뷰 → 관리자 검토
- 공지사항 조회
- 고객센터 문의 — 문의 유형 선택, 답변 알림

- 포인트 사용 — 최소 사용 단위는 `policy.md` 참조 / TBD

- 쿠폰함 조회 — 보유/만료 분리

## 운영 (관리자)

- 회원 목록/제재 관리 — 정지/강제 탈퇴
- 신고 리뷰 처리 — 검토 및 삭제
- 정산 처리 — 사장별 수익 − 수수료 (주기·규칙은 `policy.md`)
- 정산 내역 조회 (사장) — 기간별 조회
- 쿠폰 발급 — 이벤트/조건 기반
- 이벤트 생성/관리 — 대상/기간/수량 설정
- 공지사항 작성 — 즉시/예약 발송
- 문의 답변
- 약관 관리 — 버전 관리, 변경 고지

## 통계

- 매장 매출 통계 (사장) — 일/주/월별
- 마감 임박 상품 판매 현황 (사장) — 폐기 절감량
- 매장 주문 통계 (사장) — 픽업 완료율 등
- 서비스 전체 통계 (관리자) — DAU/MAU/주문량/매출
- 전체 주문 모니터링 (관리자) — 실시간
- 결제 내역 관리 (관리자) — 이상 결제 검토
