# Glossary

서비스 도메인 용어 및 영문 매핑. 정책(수치·규칙)은 [`policy.md`](policy.md) 참조.

## 사용자

### 사장 (Seller)
매장을 운영하는 점주. 사업자 인증 + 관리자 승인 후 운영 시작.

### 소비자 (Customer)
주변 마감 임박 상품을 탐색·구매·픽업하는 일반 사용자. 회원만 구매 가능.

### 관리자 (Admin)
매장 승인·회원 제재·전체 운영을 담당하는 내부 운영자.

### 주소지 (Address)
소비자가 등록하는 자기 위치 정보. 알림 반경 fallback 채널 및 미래 탐색 기준. 최대 3개. 기본 주소지 정확히 1개 (보유 0개 허용).

## 상품 / 매장

### 매장 (Store)
사장이 등록·운영하는 가게.

### 일반 상품 (Product)
정상가로 판매되는 매장의 메뉴.

### 마감 임박 상품 (ClearanceItem)
별칭: **떨이** (서비스 UI 표기). 마감 시간 전에 할인 판매하는 재고 상품. 일반 상품과 별도 엔티티로 관리.

### 마감 (Closed)
마감 임박 상품의 판매 종료 상태. 시간 도래 또는 사장 수동 마감으로 진입. 마감 임박 상품 상태 enum: `OPEN | SOLD_OUT | CLOSED`.

## 거래

### 선결제 (Prepayment)
픽업 전 앱에서 미리 결제하는 방식. 현장 결제 미지원.

### 픽업 (Pickup)
소비자가 매장을 직접 방문해 상품을 수령하는 행위. 배달 미지원.

### 노쇼 (NoShow)
소비자가 픽업 시간 내에 매장을 방문하지 않은 상태.

### 정산 (Settlement)
사장별 판매 수익에서 플랫폼 수수료를 제외한 금액을 주기적으로 지급하는 절차.

## 혜택

### 포인트 (Point)
결제 시 적립·사용되는 사용자 혜택.

### 쿠폰 (Coupon)
관리자/이벤트로 발급되는 할인 혜택.

### 즐겨찾기 (Favorite)
소비자가 매장을 등록해두는 기능.

## 알림

### 푸시 알림 (PushNotification)
서버 → 사용자 디바이스로 발송되는 알림. 마감픽의 **소비자·사장 클라이언트 (PWA)** 에 한해 **FCM (Firebase Cloud Messaging)** 으로 발송. PWA 환경에서 Web Push 메커니즘으로 수신. 관리자에게는 푸시 발송 X.

### 반경 알림 (ProximityNotification)
마감 임박 상품 등록 시 설정 거리 이내 소비자에게 발송되는 푸시 알림.
